# Elevare - Dashboard Interno

Dashboard interno para organização de estratégias, planos, mensagens, pacotes e rotinas da Agência Elevare.

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

O site estará disponível em `http://localhost:3000`

## Build

```bash
npm run build
```

## Funcionalidades

- Página inicial com visão geral
- Planejamento Social Media
- Mensagens Prontas (editáveis)
- Gerenciamento de Pacotes
- Ideias de Conteúdo
- Cadastro de Clientes
- Checklists editáveis
- Documentos e links internos

Todos os dados são salvos no localStorage do navegador.
